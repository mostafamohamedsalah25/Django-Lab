# ITIAN Management System

A comprehensive Django-based web application for managing courses and trainees. This project features a robust server-side rendered UI styled with a modern Dark Theme using Tailwind CSS, and a fully functional RESTful API built with Django REST Framework (DRF).

## 🚀 Features

- **Trainee Management:** Complete CRUD (Create, Read, Update, Delete) operations for Trainees.
- **Course Management:** Simplified course assignment with relational database mapping (Foreign Keys).
- **RESTful API:** Dedicated API endpoints for seamless integration with frontend frameworks or mobile apps.
- **Secure Authentication:** Implements Session and Basic Authentication to protect sensitive operations.
- **Role-Based Access:** Read-only access for guests, while creation, updates, and deletion require authenticated users.
- **Media Handling:** Supports profile image uploads for trainees.
- **Modern UI:** Responsive, professional Dark Mode interface built with local Tailwind CSS.

## 🛠️ Tech Stack

- **Backend:** Python, Django (6.0.4)
- **API:** Django REST Framework (DRF)
- **Database:** MySQL
- **Frontend/Styling:** Django Templates, Tailwind CSS (Node.js/npm build pipeline)
- **Image Processing:** Pillow

## 📋 Prerequisites

Before you begin, ensure you have met the following requirements:
- Python 3.10+
- Node.js & npm (for Tailwind CSS compilation)
- MySQL Server running locally
